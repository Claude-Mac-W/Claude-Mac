## Overview

Claude Professional Suite is a complete AI assistant platform with extended context, artifact creation, and project organization for professional AI-assisted work.

Extended context handles long documents, codebases, and detailed conversations that exceed the limits of standard AI interactions. The extended context enables comprehensive analysis of full documents without truncation.

Artifacts create standalone documents, code files, and visualizations within conversations that can be iterated and exported, producing tangible outputs from conversational interactions.

Project organization groups related conversations with shared context and instructions, creating focused workspaces for different professional domains or ongoing projects.

Claude Professional Suite is the AI assistant platform for professionals who need thoughtful AI assistance with the context depth, output capability, and organizational tools that professional knowledge work demands.

## Performance & Integration

Claude is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Claude processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
